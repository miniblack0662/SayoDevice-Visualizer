## Setup
```bash
pip install -r requirements.txt
python app.py
```
And you will be directed to http://localhost:7027

## OBS Setup
- Add Source - **Browser Source**
- URL: `http://localhost:7027/overlay`

## Control Panel
- Key Bindings
- Adjust Position

## Default Keys
| Action | Default Key |
|---------|--------|
| Left key  | ESC |
| Middle key | Z  |
| Right key  | X  |
